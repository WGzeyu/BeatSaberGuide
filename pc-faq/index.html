  • 为什么加载下载Mod缓慢甚至报错？ModAssistant在加载/下载时报错怎么办？

    通常10秒就能装完全部常用Mod。但是，服务器在国外，中国国际线路经常拥塞，慢到下不动也很正常。

    推荐尝试以下方案解决Mod下载问题：
    1. 下载增强版ModAssistant，打开后在左下方将Mod下载源切换为国内源。（不建议使用其他源）

  • 为什么我装的Mod不生效？装不上Mod怎么办？

    在Mod安装完毕，但游戏中找不到Mod时，才可按本篇教程操作。
    如果你的ModAssistant压根就不给装Mod，请看“为什么加载下载Mod缓慢甚至报错？”

    如果你在2022年3月以前安装的Mod能生效，只有在这之后才无法生效 (1.20以上)，请先查看上一条解答。

    安装Mod之前需要确保以下几项：
    1. 确保游戏安装路径中不存在任何中文或特殊字符，否则将导致绝对装不上Mod（不是盘符 ，你需要确保ModAssistant的“选项”界面中显示的游戏安装路径的整个框中不得存在任何中文字符）；
    2. 确保ModAssistant左下角选择的游戏版本，以及设置的游戏目录与你的游戏相同；
    3. 安装Mod前至少要启动过一次安装/更新后的游戏，退出后才能安装Mod，显示安装完成再启动游戏；
    4. ModAssistant推荐使用增强版的国内或国际源，不推荐使用增强版的包子源与原版。

    如果Mod就是不生效，请尝试以下解决方法：
    1. 进入游戏安装目录，右键以管理员身份运行IPA(.exe)。

    如果Mod仍然没有生效，请尝试以下解决方法：
    1. 按照本网页中的“如何彻底重装游戏并恢复数据？”的步骤来足够彻底地重装一次游戏。

    卸载重装很可能解决不了问题，因为不够彻底，请按教程给出的重装步骤来才做，这样才能保证解决问题。
    不要惧怕重装，教程给出的是最快、最彻底的备份重装并恢复所有数据的方案，不会损失任何数据。

    如果仍然遇到问题装不上Mod，请在上方导航栏中找到群号，加群提问。

  • 如何彻底重装游戏并恢复数据？（最快且最彻底的备份重装方案）

    只有在Mod安装完成之后出现问题，或者是游戏自身的问题，才能按照这篇教程操作。
    如果你的ModAssistant报错，或者是网页报错，重装游戏是解决不了的！

    如果你安装了360之类的第三方杀软，建议在重装前将其更换为系统自带杀软（Windows Defender），
    以下是最快且最彻底的重装游戏并恢复全部数据方案：

    1. 退出正在运行的ModAssistant（如未打开请忽略）；
    2. 找到游戏目录，进入安装路径的上一级文件夹，将游戏安装文件夹的名字改掉，随便改，有变化就行；
    3. 在Steam中找到Beat Saber，右键、属性、本地文件、验证游戏完整性，此时将会重新下载安装；
    4. 等待安装完成后，先运行一次游戏然后再退出游戏；
    5. 打开ModAssistant安装Mod（请使用增强版的国内或国际源，不推荐使用增强版的包子源与原版）；
    6. 启动游戏看看，如果仍然有问题，试试在ModAssistant的选项界面中找到游戏存档，备份后删除；
    7. 如果一切正常，退出游戏，接下来开始恢复数据：
    8. 从原来更名文件夹中删除该文件："Beat Saber\UserData\Beat Saber IPA.json"；
    9. 将以下目录从原来更名文件夹中移动到现在的游戏目录
         Beat Saber \ Beat Saber_Data \ CustomLevels     (自定义歌曲)
         Beat Saber \ CustomAvatars     (人物模型)
         Beat Saber \ CustomNotes     (方块模型)
         Beat Saber \ CustomPlatforms     (场景模型)
         Beat Saber \ CustomSabers     (光剑模型)
         Beat Saber \ UserData     (Mod设置)

    （注：移动比复制快很多；千万不要将Beat Saber_Data整个移动回去，只能移动它里面的CustomLevels）

  • 如何降级到指定的游戏版本？

    【新降级方案】（限1.29.1版本，暂时有效）

    降级教程：
    　0. 关闭ModAssistant
    　1. 在Steam中找到Beat Saber
    　2. 右键 → 属性 → 测试版
    　3. 将「参与测试」右侧的「无」切换至「legacy1.29.1_unity_ver2019.4.28f1 - pre unity upgrade b……
    　4. 等待下载完成，即可完成降级

    注：请不要填写测试代码，也不要点击「检查代码」。
    理论来说，你无需再按传统方案的繁琐步骤来恢复数据，
    如果以前给1.29.1打过Mod，切换测试分支后应该会恢复。
    如果你没打过Mod，降级完成后即可按教程来安装Mod。
    这不是测试版，只是利用该功能提供了引擎更新前的正式版

    【传统降级方案】

    下载方法：
    　1. 点我进入网盘
    　2. 选择任意一个网盘，点击【进入下载】
    　3. 进入【游戏备份与懒人包】文件夹
    　4. 找到1.29.1的原版备份
    　5. 对着找到的文件右键，点击下载

    降级教程：
    　0. 确保已经通过Steam更新了游戏
    　1. 对着下载的原版备份右键
    　2. 点击解压至单独文件夹
    　3. 通过Steam打开游戏目录
    　4. 点击游戏目录的上一级目录
    　5. 更改游戏目录文件夹名称，随便改
    　6. 将原版备份解压出的文件夹移动过来
    　7. 将原版备份文件夹名称重命名为Beat Saber

    恢复数据教程：
    　1. 打开降级教程第⑤步改名的游戏目录
    　2. 删除UserData文件夹内的Beat Saber IPA
    　3. 复制以下文件夹
    　　UserData（Mod设置）
    　　CustomSabers（光剑模型）
    　　CustomPlatforms（场景模型）
    　　CustomAvatars（人物模型）
    　　CustomNotes（方块模型）
    　4. 进入新的游戏目录
    　5. 粘贴复制的那些文件夹
    　6. 在弹出的提示中，选择【替换】
    　7. 打开降级教程第⑤步改名的游戏目录
    　8. 进入Beat Saber_Data文件夹
    　9. 剪切CustomLevels文件夹
    　10. 进入新的游戏目录
    　11. 进入Beat Saber_Data文件夹
    　12. 粘贴剪切的那个文件夹
    　13. 在弹出的提示中，选择【替换】

    降级后安装Mod教程：
    　0. 确保降级后至少打开过一次游戏
    　1. 退出游戏与降级以前打开的ModAssistant
    　2. 按正常流程打开ModAssistant就可以安装Mod了

    （其实这一套步骤做下来，就是把游戏文件整个更换为旧版备份，不包含任何残留的Mod或游戏文件，只恢复以前的设置与资源文件）

  • 装上Mod打开游戏出现 Pico串流断开 与 游戏卡死 或 SteamVR崩溃 是怎么回事？

    以前23年6月出过该问题，经Pico工程师排查是串流与Mod抢占GPU资源造成，早已推送更新进行了修复。
    但是，后来又逐渐有用户反馈出现类似的问题，最近10月份出现该问题的Pico用户也是越来越多。

    SteamVR崩溃后，需点击“管理加载项”、取消屏蔽所有加载项后，才能恢复使用串流。

    2023/10/16更新：有Pico用户报告称按此步骤操作可以解决该问题
    1. 启动并连接Pico串流，让Pico串流助手启动SteamVR
    2. Pico串流助手中点击左上角的“返回”，让其断开连接，但不要退出SteamVR
    3. 启动Beat Saber，等待游戏载入完成
    4. Pico串流助手重新连接串流，即可正常进入游戏

    2023/10/16更新：另一位Pico用户报告了一份不同的解决方法
    1. 启动并连接Pico串流，让Pico串流助手启动SteamVR
    2. 点按头显电源键，让其进入息屏状态，并确认SteamVR中头显图标处于灰色待机状态
    3. 启动Beat Saber，等待游戏载入完成
    4. 点按头显电源键、点亮并戴上头显，即可正常进入游戏

    2023/10/01更新：有用户报告清除Pico串流助手的应用数据后暂时解决了该问题。

    如果没有解决，可以试试按“如何彻底重装游戏并恢复数据？”的步骤来彻底重装游戏。

    如果仍然没有解决该问题，你可以试试切换游戏版本，将【最新版 / 1.29.1】这两个版本各试一遍，
    如需降级至1.29.1，你可以点我打开降级教程，或者也可以点我打开降级教程。

    如果仍然没有解决该问题，也可以咨询VR设备厂商客服，
    或者暂时不要安装Mod，目前看来很可能是Pico串流与Mod之间产生的冲突造成的问题。

    注：如果你遇到的不是断连或卡死，而是在降级后Pico手柄扳机键不能点，请点我打开手柄问题解决方法

  • 一打开游戏就卡黑屏、卡Logo进不去，要怎么解决？

    首先需要注意，Mod需要匹配游戏版本，如果你安装了版本不匹配的Mod，请卸载掉对应不匹配的Mod。

    如果你的游戏是最新版，请先尝试降级至1.29.1，你可以点我打开降级教程，或者也可以点我打开降级教程。

    如果你装了SmartSongSuggest这个Mod，打开游戏就黑屏未响应，试试将其卸掉。

    请确保不要直接在文件夹中双击exe启动游戏，请通过Steam或Steam创建的快捷方式来启动游戏。

    如果仍然没有解决这个问题，请按“如何彻底重装游戏并恢复数据？”来彻底重装游戏。
    （注意：卸载重装可能解决不了，按教程给出步骤重装才足够彻底，请务必将游戏安装至纯英文路径）

    如果你的游戏目录中自动出现了“游侠热门单机游戏”的快捷方式，可能是系统中毒导致的，被病毒修改过的文件与Mod不兼容从而无法启动，如果遇到该情况建议重装系统，不要运行以前的任何exe可执行文件。

    注：本条解答无法解决因VR端软件问题或者SteamVR问题而导致的黑屏现象。

  • 游戏更新到1.20.0及以上版本之后更新BSIPA总是变回原来的旧版怎么办？

    1. 使用ModAssistant卸载BSIPA（ModAssistant的“选项”界面中有这个按钮）
    2. Steam验证游戏文件
    3. 使用ModAssistant安装Mod（推荐使用增强版的国内或国际源，不推荐使用增强版的包子源与原版）
    4. 启动游戏，然后退出游戏
    5. 再次使用ModAssistant安装Mod（推荐使用增强版的国内或国际源，不推荐使用增强版的包子源与原版）

  • 为什么有的Mod在ModAssistant找不到？

    说明Mod还没有更新到正式版。（不是说正版不支持，[正式版/测试版]与[正版/盗版]不是一个意思）

    着急用的话可以先手动安装测试版Mod，常用mod一般会有人维护更新的，可以在【BS新手教程】中找到手动安装Mod教程。

    一些Mod更新没那么快，游戏更新也经常会出现新Mod，可以每隔一段时间打开安装器看看。

  • 游戏刚刚更新完，Mod就消失了，怎么办？需要禁止更新吗？

    启动过一次新版游戏后，退出游戏使用ModAssistant重新安装Mod即可。（推荐使用增强版的国内或国际源，不推荐使用增强版的包子源与原版）

    注：游戏刚刚更新的几小时内可能识别不到最新的游戏版本，此时建议等待一会再安装Mod。

    我们反对禁用更新。现在Mod开发者都在紧随游戏最新版，
    虽然在游戏刚刚更新的几小时内，有可能没有Mod，但通常在一天内Mod就都会更新。
    更何况，现在更新的很多游戏版本，Mod直接完全兼容，都不用等直接就有，重装Mod就行。

    如果部分Mod暂未更新正式版，可以按【BS新手教程】中的手动安装Mod教程来安装。

  • 装完Mod以后，启动游戏点击CONTINUE就会黑屏，要怎么解决？

    如果你使用ModAssistant安装Mod之前，切换过ModAssistant的游戏版本，
    那说明是由于你安装了版本不匹配的Mod，把游戏搞坏了导致无法运行。
    安装版本匹配的Mod一般不会有问题，版本不匹配的Mod才会导致游戏运行不正常。

    请按照上方“如何彻底重装游戏并恢复数据？”的方案，彻底重装游戏。

    如果你遇到的不是这个情况，请加群提问，可在顶部导航栏中找到群号。

  • 为什么Beat Saber安装目录中找不到我刚刚装的Mod或songe-converter？

    安装Mod后运行一次游戏，才算完整安装，Mod相关文件才会出现在他该出现的位置。
    在这之前，Mod相关文件会保存在Beat Saber\IPA\Pending

    另外，新版Mod已不再附带远古格式歌曲转换工具songe-converter，如有需要可进入BS游戏资源共享中的工具目录下载。

  • 为什么ModAssistant显示“BSIPA解压失败”？

    目前我见到的该问题，全部都是由于安装Mod的过程遭到拦截导致的。
    我们一般推荐使用系统自带杀软（Windows Defender），一般不会有这种问题，查杀率也很高，
    而360这类第三方杀软经常无提示误拦截，所以出现这种情况也不奇怪。

    如果更换杀软不能解决问题，可以尝试在右上角加群提问。

  • 为什么ModAssistant左下角的版本切换中没有匹配的游戏版本？

  • 为什么歌曲格式有不同版本？如何区分？

    在Beat Saber刚刚发布测试版的时候，官方还不支持自定义歌曲，此时有玩家制作了Mod来实现自定义歌曲并定义了歌曲格式，这就是远古版本的歌曲格式，谱面文件为 .json 后缀。

    后来没多久，游戏官方就发布了支持自定义歌曲的更新并由官方定义了一个不一样的自定义歌曲格式，
    这就是现在使用的歌曲格式，谱面文件为 .dat 后缀。
    在官方定义了歌曲格式以后，即使是通过Mod来加载歌曲也都使用官方格式，远古格式早已被弃用。

    在游戏更新至1.20.0版本之后增加了新的音符类型，格式依然为 .dat ，
    含有新音符类型的谱面无法被老版本游戏读取。

    请不要通过音频文件来识别歌曲格式，游戏支持读取ogg格式音频文件，但游戏才不管文件名和后缀名，只会照着info中记录的文件名去读文件，而一些歌曲网站为了一定程度上的规避版权问题，将音频文件改了后缀名来防止你双击播放，这才导致音频文件存在不同的“格式”（只是后缀名不一样，格式是一样的）。
    如果你愿意，音频文件后缀名可以改成任意名称，只要连info中的信息也一起改掉就行。

  • 为什么不推荐使用无线或有线串流VR玩Beat Saber？

    从原理上来说，直连VR相当于你的显示器，而串流VR就相当于平板全屏播放直播画面来充当显示器，
    不管你是用Wi-Fi看直播，还是用有线、连网线看直播，直播他就是直播，和直连显示的设备比不了。

    串流会对画面进行有损编码和网络传输，而目前的编码与解码，带来的画面延迟高到以至于不适合音游，
    目前异步扭曲只能补偿头显旋转，画面内容及内容的位置延迟并不能补偿，会让人误以为是手柄有延迟。

降级教程

【新降级方案】（限1.29.1版本）

降级教程：
　0. 关闭ModAssistant
　1. 在Steam中找到Beat Saber
　2. 右键 → 属性 → 测试版
　3. 将「参与测试」右侧的「无」切换至「legacy1.29.1…
　4. 等待下载完成，即可完成降级

注：请不要填写测试代码，也不要点击「检查代码」。
理论来说，你无需再按传统方案的繁琐步骤来恢复数据，
如果以前给1.29.1打过Mod，切换测试分支后应该会恢复。
如果你没打过Mod，降级完成后即可按教程来安装Mod。
这不是测试版，只是利用该功能提供了引擎更新前的正式版

【传统降级方案】

下载方法：
　1. 点我进入网盘
　2. 选择任意一个网盘，点击【进入下载】
　3. 进入【游戏备份与懒人包】文件夹
　4. 找到1.29.1的原版备份
　5. 对着找到的文件右键，点击下载

降级教程：
　0. 确保已经通过Steam更新了游戏
　1. 对着下载的原版备份右键
　2. 点击解压至单独文件夹
　3. 通过Steam打开游戏目录
　4. 点击游戏目录的上一级目录
　5. 更改游戏目录文件夹名称，随便改
　6. 将原版备份解压出的文件夹移动过来
　7. 将原版备份文件夹名称重命名为Beat Saber

恢复数据教程：
　1. 打开降级教程第⑤步改名的游戏目录
　2. 删除UserData文件夹内的Beat Saber IPA
　3. 复制以下文件夹
　　UserData（Mod设置）
　　CustomSabers（光剑模型）
　　CustomPlatforms（场景模型）
　　CustomAvatars（人物模型）
　　CustomNotes（方块模型）
　4. 进入新的游戏目录
　5. 粘贴复制的那些文件夹
　6. 在弹出的提示中，选择【替换】
　7. 打开降级教程第⑤步改名的游戏目录
　8. 进入Beat Saber_Data文件夹
　9. 剪切CustomLevels文件夹
　10. 进入新的游戏目录
　11. 进入Beat Saber_Data文件夹
　12. 粘贴剪切的那个文件夹
　13. 在弹出的提示中，选择【替换】

降级后安装Mod教程：
　0. 确保降级后至少打开过一次游戏
　1. 退出游戏与降级以前打开的ModAssistant
　2. 按正常流程打开ModAssistant就可以安装Mod了

（其实这一套步骤做下来，就是把游戏文件整个更换为旧版备份，不包含任何残留的Mod或游戏文件，只恢复以前的设置与资源文件）
